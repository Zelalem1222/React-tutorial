# Todo App

> This is a todos list app build with Reactjs.


## Built With

- HTML
- CSS 
- Javascript
- React

## Getting Started

To get a local copy up and running follow these simple example steps.

- Clone repository in your local machine 
- cd React-Tutorial
- npm install
- npm start
